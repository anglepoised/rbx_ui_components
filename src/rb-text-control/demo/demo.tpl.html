The rb-text-control component groups simple, text-based fields that share a set of common states. Status and modifier classes can be applied to any field type and can be stacked.

The placeholder attribute should be used where this will aid usability but should not be used in place of a label element.

The autocomplete="off" attribute should not be used for password fields, but should be used for other sensitive data types (e.g. credit card information).

The autofocus attribute should only be used once per page, and only on the first field on a composed page that has a single, simple form (e.g. rb-login-form).
