The rg-header component should be used across all products. It consists of:

  • A mandatory logo
  • An optional rb-nav-bar subcomponent
  • An optional rb-user subcomponent
  • The rg-header-item may contain an href or rg-ui-sref (resolved to ui-sref) property and an optional target attribute.

It must be enclosed within the primary page rb-header element.
