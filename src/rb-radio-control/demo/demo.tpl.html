A radio control must always display more than one radio button and label pair.

When using the `RadioControl--row` modifier, exactly two radio button and label pairs.

We may not need (or be able to use) all of these states as we're swapping out native controls for custom styled controls.

Required: "If any radio button in a same named group of radio buttons has the required attribute set, the user must select one of the radio buttons in that group, but not necessarily the one that has the attribute set. The one occurrence of the required attribute is applied to the whole group of same named radio buttons." See: http://www.wufoo.com/html5/types/15-radio.html

Pressed: distinct from checked. Click and hold but don't release to activate.

Focused: distinct from pressed. Tab to control with keyboard, but don't hit space to activate.
