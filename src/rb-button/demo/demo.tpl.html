The rb-button component defines button styles for <button> and <a> elements.

Solid buttons are the default style. Add an outline="yes" attribute for outline buttons. When in doubt, use a standard, solid button.

Block Modifier

The block modifier should be used to make a button full-width. It adds width: 100%;, changes the display from inline-block to block, and centers the button text.

Action Modifiers

Standard buttons are blue. Action modifiers change the colour of buttons and can be used to help indicate the likely consequences of different types of action.

Positive

Positive buttons are green. They indicate a successful or positive action.

Positive Button Positive Button
Warning

Warning buttons are orange. They indicate that care should be taken with this action.

Warning Button Warning Button
Danger

Danger buttons are red. They indicate a potentially destructive action.
