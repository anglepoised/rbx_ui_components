.SystemMessage

API

  • title: Title atop the component.
  • message: System message in layman's terms.
  • technical-details: Technical message for more information.
  • on-dismiss: An expression evaluated when dismiss button is clicked.
  • contact [optional]: Contact email address.
  • modifier [optional]: Change the mode of the system message. Can be danger or positive.
  • dismiss-message [optional]: Custom message for the dismiss button. Defaults to "Okay I Understand".
